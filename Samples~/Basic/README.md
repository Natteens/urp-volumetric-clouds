# Basic Sample

Este é um exemplo básico para demonstrar o uso do URP Volumetric Clouds.

## Como usar

1. Importe esta amostra através do Package Manager
2. Abra a cena de exemplo
3. Execute para ver o URP Volumetric Clouds em ação

- Cenas básicas de exemplo
- Prefabs simples
